# Sanskriti — Jadavpur University Cultural Fest

The official website for Sanskriti, Jadavpur University's annual cultural festival. Features event listings, countdown timer, photo gallery, and an admin dashboard for event management.

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Framer Motion
- React Router

## Installation

```bash
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install
npm run dev
```

The app runs at `http://localhost:8080`.
